## Week summaries
- h0: First captures with tshark/Wireshark, DNS/TCP/HTTP flow, TLS handshake, and OSI/TCP-IP layer mapping.
- h1: Kali install notes, network blocking with iptables, pcap analysis, and TCP handshake walkthroughs.
- h2: Pyramid of Pain and Diamond Model summaries, Apache log analysis, Nmap -A scripting traces, and log evasion tweaks.
- h3: Report on selected DEF CON talks with key takeaways from RF, malware, Tor, and IoT privacy themes.
- h4: RFID inventory and threat reflection, APDU structure overview, and MIFARE Classic backdoor news.
- h5: Evilginx2 local phishing demo and Mininet/SDN lab notes including SYN flood observations.
- h6: WiFiChallengeLab Docker setup, WiFi recon and attack tooling, and security reflections.
- h7: WebSDR listening, rtl_433 and URH workflows, and RF signal demodulation/analysis.
